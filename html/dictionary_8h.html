Loads a subset of a dictionary into a provided buffer.

This function parses dictionary data from a byte array and populates an array of DictionaryEntry structures. It avoids heap usage by using a pre-allocated buffer.

Parameters
dataPointer to the raw dictionary byte array.
sizeThe size of the dictionary byte array.
offsetThe starting byte offset for the dictionary subset.
child_countThe number of dictionary entries to load. A value of -1 indicates the root entry.
out_entriesA pre-allocated array where the loaded entries will be stored.
out_countPointer to a variable to store the number of loaded entries.
Returns
true if the subset was loaded successfully, false otherwise.
